# Сайт кофейни "Зёрнышко"

Добро пожаловать! Это сайт - учебный проект, кофейни "Зёрнышко".


## Как запустить сайт у себя на компьютере

Так как проект не использует сборщики (например, Create React App или Vite), запустить его очень просто:

1.  Скачайте или клонируйте этот репозиторий на свой компьютер.
2.  Найдите в папке проекта файл `index.html` (или другой главный HTML-файл).
3.  **Дважды щелкните** по этому файлу, чтобы открыть его в браузере.

*Примечание: Если вы хотите использовать локальный сервер для разработки (например, чтобы работал AJAX), вы можете установить его с помощью Python:*

*   Откройте терминал в папке проекта.
*   Запустите команду: `python -m http-server` (для Python 3) или `python -m SimpleHTTPServer` (для Python 2).
*   Сайт будет доступен по адресу [http://localhost:8000](http://localhost:8000).


## Технологии

*   **HTML5** — структура страницы.
*   **CSS3 / SASS** — стилизация и адаптивный дизайн.
*   **JavaScript (ES6+)** — интерактивность.

## Структура проекта

*   **`index.html`** — главный HTML-файл сайта.
*   **`main.js`** — Основной файл с 
*   **`JavaScript`** -кодом. Управляет интерактивностью.
*   **`_vars.scss`** — Вспомогательный файл с переменными (палитра цветов, сетка, шрифты). Используется в styles.scss.
styles.scss — Исходный файл стилей, написанный на препроцессоре SASS. 
*   **`style.css`** — файл со стилями (CSS) для оформления сайта.
*   **`/images`** (или `/img`) — папка, в которой хранятся все изображения (фото кофе, фоновые картинки).
*   **`style.css.map`**  и **`index.css.map`**  — Карты стилей. Помогают находить исходный код в DevTools, если включен режим сжатия (minify).
